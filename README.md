# IchigoRuby

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

IchigoRuby is a web-based Ruby interpreter that allows you to run Ruby scripts and view the output directly in the browser.

## Demo
You can access the live demo at [https://code4fukui.github.io/IchigoRuby/](https://code4fukui.github.io/IchigoRuby/).

## Features
- Web-based Ruby interpreter
- Run Ruby scripts directly in the browser
- Provides a simple and intuitive interface

## Requirements
- Web browser with modern JavaScript support

## Usage
1. Open the [IchigoRuby demo page](https://code4fukui.github.io/IchigoRuby/).
2. Write your Ruby script in the left-hand textarea.
3. Click the "RUN" button to execute the script.
4. The output will be displayed in the right-hand textarea.
