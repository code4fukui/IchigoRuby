# IchigoRuby

IchigoRubyは、Webブラウザ上で直接Rubyスクリプトを実行し、その出力を表示できるWebベースのRuby実行環境です。

## デモ
[https://code4fukui.github.io/IchigoRuby/](https://code4fukui.github.io/IchigoRuby/)で実際に試すことができます。

## 機能
- Webブラウザ上でRubyスクリプトを実行できる
- シンプルで直感的なユーザーインターフェース

## 必要環境
- 最新のJavaScriptをサポートするWebブラウザ

## 使い方
1. [IchigoRubyのデモページ](https://code4fukui.github.io/IchigoRuby/)を開きます。
2. 左のテキストエリアにRubyスクリプトを記述します。
3. 「RUN」ボタンをクリックするとスクリプトが実行されます。
4. 右のテキストエリアに出力が表示されます。
