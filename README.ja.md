# IchigoRuby

IchigoRubyは、ブラウザ上で直接Rubyスクリプトを実行し、出力を確認できるWebベースのRubyインタプリタです。

## デモ
ライブデモは [https://code4fukui.github.io/IchigoRuby/](https://code4fukui.github.io/IchigoRuby/) からアクセスできます。

## 機能
- WebベースのRubyインタプリタ
- ブラウザ上で直接Rubyスクリプトを実行可能
- シンプルで直感的なインターフェース

## 必要条件
- モダンなJavaScriptをサポートするWebブラウザ

## 使い方
1. [IchigoRubyのデモページ](https://code4fukui.github.io/IchigoRuby/)を開きます。
2. 左側のテキストエリアにRubyスクリプトを入力します。
3. 「RUN」ボタンをクリックしてスクリプトを実行します。
4. 右側のテキストエリアに出力結果が表示されます。
